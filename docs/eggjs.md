# 介绍 Egg.js

Egg.js 官网: [https://eggjs.org](https://eggjs.org)

Egg.js 是一个基于 Koa 开发的 Node.js 框架

Egg 奉行『约定优于配置』，按照一套统一的约定进行应用开发，团队内部采用这种方式可以减少开发人员的学习成本
